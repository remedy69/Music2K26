Replace your repo files with the included bot.py, requirements.txt, and nixpacks.toml.

Railway variables:
- DISCORD_TOKEN = your bot token
- YOUTUBE_COOKIES = optional but strongly recommended for YouTube reliability
- COMMAND_PREFIX = optional, default !

Discord Developer Portal:
- Enable MESSAGE CONTENT INTENT
- Enable SERVER MEMBERS INTENT only if you later add DJ/permission features tied to member role checks
- Invite bot with scopes: bot, applications.commands
- Give permissions: Send Messages, Embed Links, Connect, Speak, Use Slash Commands

Commands included:
- /play /pause /resume /skip /stop
- /queue /nowplaying /volume /loop
- /autoplay /mode247 /filter /panel
- /saveplaylist /loadplaylist /deleteplaylist /playlists_cmd
- /shuffle /remove /clear

Notes:
- Spotify/Apple Music direct resolution is not truly implemented because that requires external APIs or a Lavalink stack.
- SoundCloud links should usually work through yt-dlp.
- The bot stores playlists and guild settings in ./data.
